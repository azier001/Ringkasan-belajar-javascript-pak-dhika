> **ğŸ’¡ Konsep:** Map mengubah setiap item menjadi bentuk baru dengan fungsi yang sama

### ğŸ“Š Reduce - Menggabungkan Data

> **ğŸ’¡ Konsep:** Reduce menggabungkan semua item menjadi satu nilai dengan cara akumulasi
