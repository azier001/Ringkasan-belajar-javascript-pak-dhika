### ğŸ“„ String

Menggunakan **FOR OF** untuk menampilkan setiap karakter pada string:

```javascript

  console.log(n);
```

> **Penjelasan:** Variabel `n` merepresentasikan setiap karakter pada string `nama` dalam setiap pengulangan.

> **Penjelasan:**

### ğŸ“ Penjelasan

**FOR IN** hanya digunakan untuk **enumerable** (properti pada object).

> **âš ï¸ Penting:** FOR OF tidak bisa digunakan pada Object. Untuk looping pada object gunakan FOR IN.

> **Penjelasan:**
> - `m` â†’ merepresentasikan nama properti pada object
> - `mhs[m]` â†’ mengambil value dari setiap properti dalam object
> - Variabel `m` dijadikan index dengan kurung siku `[]`

## ğŸ“‹ Kesimpulan

| Loop Type | Digunakan untuk | Contoh Penggunaan |
|-----------|----------------|-------------------|
| **FOR OF** | Iterable objects (String, Array, NodeList, Arguments) | `for (const item of array)` |
| **FOR IN** | Enumerable properties (Object) | `for (const key in object)` |

### ğŸ¯ Tips Penting:

- âœ… **FOR OF** â†’ Untuk iterasi nilai (values)
- âœ… **FOR IN** â†’ Untuk iterasi properti (properties)
- âœ… **FOR OF** tidak bisa digunakan pada plain objects
- âœ… **FOR IN** khusus untuk object properties
- âœ… Gunakan `entries()` dengan FOR OF jika membutuhkan index
