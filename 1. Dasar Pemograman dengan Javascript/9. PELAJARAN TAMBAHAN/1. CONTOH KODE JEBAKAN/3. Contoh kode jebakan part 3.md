### ğŸ§  12. `for-in` vs `for-of`

```javascript
const arr = ["a", "b", "c"];
for (const i in arr) {
  console.log(i); // 0, 1, 2
}
for (const i of arr) {
  console.log(i); // a, b, c
}
```

**â— Penjelasan:**

* `for-in` mengiterasi *key* atau *index* (untuk objek atau array).
* `for-of` mengiterasi *value*.

**âœ… Solusi:**
Gunakan `for-of` untuk array ketika kamu butuh nilainya langsung.
