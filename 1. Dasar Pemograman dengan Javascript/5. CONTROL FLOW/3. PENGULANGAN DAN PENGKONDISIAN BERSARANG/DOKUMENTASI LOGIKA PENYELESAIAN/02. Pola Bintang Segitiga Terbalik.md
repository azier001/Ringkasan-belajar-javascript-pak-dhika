## ğŸ”„ Apa itu Loop?

**Loop** adalah cara untuk mengulang suatu aksi berkali-kali.

**Contoh Sederhana:**
```javascript
for (let i = 1; i <= 3; i++) {
  console.log("Halo " + i)
}
```

**Output:**
```
Halo 1
Halo 2
Halo 3
```

**Penjelasan:**
- `let i = 1` â†’ Mulai dari 1
- `i <= 3` â†’ Jalan sampai 3
- `i++` â†’ Naik 1 setiap kali
- Loop berjalan **3 kali**

<a name="apa-itu-nested-loop"></a>
## ğŸ” Apa itu Nested Loop?

**Nested Loop** adalah **loop di dalam loop**.

**Visualisasi:**
```

    â†“

```

**Contoh:**

**Output:**

> ğŸ’¡ **Poin Penting:** Loop luar mengontrol **baris**, loop dalam mengontrol **kolom** atau **isi baris**.

### Bagian-bagian Loop

#### 1ï¸âƒ£ MULAI (`j = i`)
- Menentukan **dari mana mulai**
- Tidak mempengaruhi **berapa kali** loop jalan
- Seperti: "Aku mulai hitung dari angka i"

- Seperti: "Aku terus hitung sampai j mencapai 5"

#### 3ï¸âƒ£ LANGKAH (`j++`)
- Menentukan **seberapa cepat** j bergerak
- Biasanya naik 1

<a name="tips-untuk-pemula"></a>
## ğŸ’¡ Tips untuk Pemula

### âœ… DO (Lakukan)

1. **Mulai dari yang sederhana**
   - Buat loop luar dulu
   - Tes dengan `console.log(i)`
   - Baru tambah loop dalam

2. **Gunakan nama variabel jelas**
   ```javascript

   let str = ''    // âš ï¸ Kurang jelas
   ```

3. **Debug step by step**
   - Tambah `console.log` di banyak tempat
   - Lihat nilai variabel setiap langkah

4. **Gambar dulu di kertas**
   - Visualisasikan polanya
   - Hitung manual berapa kali loop harus jalan

5. **Gunakan cara paling intuitif**
   ```javascript
   for (let j = i; j <= 5; j++)  // âœ… Mudah dipahami
   ```

---

### âŒ DON'T (Hindari)

1. **Jangan langsung kompleks**
   - Jangan langsung nested loop 3 level
   - Mulai dari 2 level dulu

2. **Jangan abaikan scope variabel**
   ```javascript
   let string = ''  // âŒ Di luar = menumpuk
   for (let i...) {
     let string = ''  // âœ… Di dalam = reset
   }
   ```

3. **Jangan lupa trace manual**
   - Jangan asal coba-coba
   - Pahami kenapa kode bekerja

4. **Jangan takut salah**
   - Error adalah bagian dari belajar
   - Setiap error adalah pelajaran
